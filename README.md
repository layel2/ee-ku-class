# EE KU Course Schedule Website
**Web : [https://ee-course.layel2.com/](https://ee-course.layel2.com/)**

Class management system for Electrical Engineering @KU

## Deployment
- Deploy DB (postgresql)
- create `.env` file
```
DB_HOST=
DB_PORT=
DB_NAME=
DB_USER=
DB_PASS=
```
- run `docker-compose up`
